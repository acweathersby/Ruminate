# RUMINATE NOTES

## Commands

### Run Frontend Dev Environment

```zsh
$ npm run dev:fe
```

### Test Frontend

```zsh
$ npm run test:fe
```

### Run App Dev Environment

```zsh
$ npm run dev:app
```

### Update Markdown Token Parser

```zsh
$ npm run build:md
```

### Build App

#### Debug

```zsh
$ npm run build:debug
```

#### Release

```zsh
$ npm run build:debug
```
